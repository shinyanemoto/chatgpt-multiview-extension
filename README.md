# ChatGPT Multi-View Extension

Chrome拡張機能でChatGPTを2x2、または1+3のレイアウトで4枚同時に表示・操作するためのツールです。

## 概要

この拡張機能は、親ウィンドウ（コントローラー）を操作することで、4つのChatGPTポップアップウィンドウをタイル状に整列・追従させます。
ChatGPTのWeb版はセキュリティ上の理由（`X-Frame-Options: DENY`）からiframeへの埋め込みができないため、複数の独立したウィンドウを座標制御することでマルチビューを実現しています。

## 特徴

- **自動整列**: 親ウィンドウを移動・リサイズすると、4つのChatGPTウィンドウが自動的にタイル状に追従します。
- **2つのレイアウト**:
  - **2x2**: 均等な4分割表示
  - **1+3**: 左側に大きなメイン画面、右側に3つのサブ画面
- **一括操作**: すべての画面のリフレッシュ、再整列、一括終了が可能です。
- **Mac / Chrome 最適化**: ポップアップ形式（ツールバーなし）で、広く画面を使えます。

## インストール手順

1. このリポジトリをローカルにクローンまたはダウンロードします。
2. Google Chrome を開き、 `chrome://extensions/` にアクセスします。
3. 右上の「デベロッパー モード」をオンにします。
4. 「パッケージ化されていない拡張機能を読み込む」をクリックし、このフォルダを選択します。

## 使い方

1. 拡張機能バーの「ChatGPT Multi-View」アイコンをクリックします。
2. 親ウィンドウ（コントローラー）が開くと同時に、4つのChatGPTウィンドウが生成されます。
3. 親ウィンドウの上部バーをドラッグして移動したり、端をドラッグしてリサイズすると、子ウィンドウが追従します。
4. レイアウト切替ボタンで好みの配置に変更できます。
5. 親ウィンドウを閉じると、すべての子ウィンドウも自動的に閉じます。

## 技術的制限・注意点

- **追従の遅延**: Chrome拡張機能にはウィンドウ移動の連続イベントがないため、100ms〜200ms間隔のポーリングで位置を監視しています。そのため、移動中に少し遅れ（カクつき）が発生しますが、マウスを離せば正確に整列します。
- **ログイン**: 初回は各ウィンドウでログインが必要な場合があります。一度ログインすれば、すべてのウィンドウでセッションが共有されます。
- **最小サイズ**: ウィンドウを小さくしすぎると、ChatGPTのUIが崩れたり、Chromeの最小ウィンドウ幅制限により整列が乱れることがあります。
- **macOSの挙動**: 親ウィンドウをフルスクリーンにすると、子ウィンドウの制御ができなくなるため、通常ウィンドウの状態でリサイズして使用することを推奨します。
